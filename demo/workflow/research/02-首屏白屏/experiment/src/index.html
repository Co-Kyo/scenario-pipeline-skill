💡 关键观察

  • CSR：白屏时间 = HTML下载 + JS下载 + JS执行 + 首次渲染。所有逻辑在客户端完成，首屏最慢。
  • SSR：HTML 已包含内容，FCP 极快。但 Hydration 需加载完整 JS bundle，TTI 可能滞后。瓶颈在服务端渲染耗时和 JS bundle 大小。
  • SSG：构建时预生成 HTML，CDN 直接返回，TTFB 最快。适合静态内容，但数据新鲜度受限于构建频率。
  • 缓存命中：强缓存命中时跳过网络请求，首屏时间接近 0。这是 CDN + HTTP 缓存协作的核心价值。
  • 网络影响：3G 下差异被放大——CSR 白屏可达数秒，SSR 仍能快速显示内容骨架。
