## 学习路径

```
Step 1: A1 渲染管线 (6/8, 累积 15.0)
  → 所有渲染/性能命题的底层基础
  → 验证：画出完整 CRP 流程图

Step 2: A4 DevTools (4/8)
  → 所有性能诊断的实操入口
  → 验证：独立用 Performance + Memory 面板定位问题

Step 3: A8 Web Vitals (4/8, 累积 6.0)
  → 性能优化的目标定义和度量基准
  → 验证：说出 LCP/INP/CLS 定义和阈值

Step 4: A2 事件循环 (3/8, 累积 5.0)
  → 理解主线程调度是解决卡顿的前提
  → 验证：画出事件循环模型图

Step 5: A6 HTTP 缓存 (3/8, 累积 5.0)
  → 网络层优化基础
  → 验证：解释强缓存与协商缓存完整流程

Step 6: A9 IntersectionObserver (3/8)
  → 虚拟列表和懒加载的核心 API
  → 验证：实现图片懒加载组件

Step 7: A13 CSS contain (3/8)
  → 渲染优化中常被忽视的 CSS 手段
  → 验证：解释 contain 三种值的区别
```
