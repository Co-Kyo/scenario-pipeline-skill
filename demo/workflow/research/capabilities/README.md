## 学习路径

### 第 1 步：A8-DevTools性能分析（覆盖 6/8 命题）
- 学习内容：Performance面板录制分析、Memory面板堆快照、Lighthouse审计、PerformanceObserver API
- 验证：能独立用 DevTools 定位一个页面的性能瓶颈

### 第 2 步：A1-浏览器渲染管线（覆盖 4/8 命题）
- 前置：A8
- 学习内容：CRP全链路 DOM→CSSOM→Style→Layout→Paint→Composite
- 验证：能分析一个万级 DOM 页面的渲染瓶颈

### 第 3 步：A4-事件循环与任务调度（覆盖 4/8 命题）
- 前置：A1
- 学习内容：宏任务/微任务、rAF/rIC调度、长任务检测与拆分
- 验证：能用 rIC 拆分一个长任务并验证 INP 改善

### 第 4 步：A2-DOM节点生命周期（覆盖 3/8 命题）
- 前置：A1
- 学习内容：节点创建/挂载/卸载/回收、Detached DOM、事件监听器泄漏
- 验证：能用 Heap Snapshot 定位 DOM 泄漏

### 第 5 步：A5-HTTP缓存协议（覆盖 2/8 命题）
- 学习内容：强缓存/协商缓存、ETag/Last-Modified、缓存策略设计
- 验证：能为一个项目设计三层缓存策略

### 第 6 步：A7-资源加载策略（覆盖 3/8 命题）
- 学习内容：preload/prefetch/lazy loading/fetchpriority
- 验证：能用 Lighthouse 识别并优化资源加载瓶颈

### 第 7 步：A6-CSS布局与合成层（覆盖 2/8 命题）
- 前置：A1
- 学习内容：contain/will-change/content-visibility、图层提升
- 验证：能用 Rendering 面板分析合成层问题

### 第 8 步：A3-V8 GC机制（覆盖 2/8 命题）
- 前置：A2
- 学习内容：分代GC、WeakRef/WeakMap、内存泄漏排查
- 验证：能用 Memory 面板拍摄堆快照并分析 GC 压力
