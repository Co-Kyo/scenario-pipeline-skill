## 字段说明

| 字段 | 含义 |
|------|------|
| `做什么` | 具体动作（降低认知门槛） |
| `你会看到什么` | 预期结果 |
| `这说明了什么` | 观察→知识连接 |
| `接下来去哪` | 指向产出文件的精确路径 |
| `做到才算过` | 二值验证标准 |
