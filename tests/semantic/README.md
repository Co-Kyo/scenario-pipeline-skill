# Layer 3 语义验证

## 设计

Agent 框架本身就是 LLM，可以直接充当测试执行器。

```
Python 测试 → 定义评估标准 → spawn 子 agent → 子 agent 评估 → 返回结果 → Python 断言
```

## 执行方式

```bash
# 运行语义验证（需要 agent 框架支持）
pytest tests/semantic/ -v
```

## 评估维度

| 测试 | 评估内容 | 评判标准 |
|------|----------|----------|
| 命题质量 | 命题是否适合目标经验水平 | 专业性、难度合理性 |
| 能力描述准确性 | 能力描述是否准确 | 技术正确性、完整性 |
| 学习阶梯合理性 | 学习路径是否合理 | 渐进性、依赖正确性 |

## 注意

- Layer 3 是非确定性的，同一测试可能多次运行结果不同
- 建议多次运行取多数通过
- 失败时需要人工审查
